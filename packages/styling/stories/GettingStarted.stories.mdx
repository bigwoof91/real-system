# Getting Started

Welcome to the Real System for React-based web applications!

<br />
<br />

## Installation & Setup

1. Install `@realsystem`

```bash
$ @todo
```

2. Install peer dependencies

```bash
$ npm i --save react react-dom
# or
$ yarn react react-dom
```

```json copy=false
These are `@realsystem` peer dependencies

"peerDependencies": {
    "react": "^17.0.2",
    "react-dom": "^17.0.2",
}
```

3. Setup styles and theming

```jsx
// index.js

import { GlobalStyles, ThemeProvider } from '@todo';

ReactDOM.render(
  <React.StrictMode>
    <GlobalStyle />
    <ThemeProvider>...</ThemeProvider>
  </React.StrictMode>,
  document.getElementById('root')
);
```

> The `GlobalStyles` component uses `styled-component`'s `createGlobalStyles` utility. It is responsible for font smoothing, setting the `html` base font-size (`.625rem`) and normalizing styles for browsers via `normalize.css`

<br />

### _**You're all set!**_
