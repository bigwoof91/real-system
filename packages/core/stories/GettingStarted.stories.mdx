# Getting Started

## Installation & Setup

### Peer dependencies

To start using Real System, you must first install the following third party dependencies:

```json copy=false

"peerDependencies": {
    "react": "^18.2.0",
    "react-dom": "^18.2.0",
}
```

```bash
# npm
$ npm install --save react react-dom

# yarn
$ yarn add react react-dom
```

### Real System Packages

Every barrelled import in Real System Core can also be installed individually.For instance,
you could install `@real-system/button` and simply import `Button` from that package.
However, we recommend using the Core package for ease of maintenance; only fall back to the
individual packages if you need to pin a certain API/component to a specific version.

```bash
# npm
$ npm install --save @real-system/core

# yarn
$ yarn add @real-system/core
```

### Theme Provider

```tsx
// index.js
import React, { ReactDOM } from 'react';

import { RealSystemProvider } from '@real-system/core/styled-library';
import { App } from './App';

ReactDOM.render(
  <React.StrictMode>
    <RealSystemProvider>
      <App>
    </RealSystemProvider>
  </React.StrictMode>,
  document.getElementById('root')
);
```

> `RealSystemProvider` provides the `theme` object to all children via React Context but it also renders `GlobalStyles` which uses `emotion`'s `Global` utility and is vital for Real System's look and feel. In short, it is responsible for
>
> - normalizing styles for browsers via `normalize.css`,
> - font smoothing,
> - setting the `html` base font-size (`.625rem`) and outline-color (`blue-400`)
> - setting the `body` font properties and background-color

## Usage

### Real System Components

You can use the Real System components so long is they are nested within Theme context.

```tsx
// index.js
import React, { ReactDOM } from 'react';

import { RealSystemProvider } from '@real-system/core/styled-library';
import { App } from './App';

ReactDOM.render(
  <React.StrictMode>
    <RealSystemProvider>
      <App>
    </RealSystemProvider>
  </React.StrictMode>,
  document.getElementById('root')
);

// App.js
import { Box } from '@real-system/core/box';

const App = () => (
  <Box margin={2} backgroundColor="white">...</Box>
);
```
