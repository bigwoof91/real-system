## Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom 
# yarn
$ yarn add react react-dom 

# install button-primitive

# npm
$ npm install @real-system/button-primitive
# yarn
$ yarn add @real-system/button-primitive
```
