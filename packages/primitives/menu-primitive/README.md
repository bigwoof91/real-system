## Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom 
# yarn
$ yarn add react react-dom 

# install menu-primitive

# npm
$ npm install @real-system/menu-primitive
# yarn
$ yarn add @real-system/menu-primitive
```
