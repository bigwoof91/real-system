## Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom 
# yarn
$ yarn add react react-dom 

# install dialog-primitive

# npm
$ npm install @real-system/dialog-primitive
# yarn
$ yarn add @real-system/dialog-primitive
```
