## Usage

```bash
# install peer dependencies

# npm
$ npm install react react-dom 
# yarn
$ yarn add react react-dom 

# install react-laag-library

# npm
$ npm install @real-system/react-laag-library
# yarn
$ yarn add @real-system/react-laag-library
```
