# Tokens

> **Design tokens are the visual design atoms of the design system — specifically, they are named entities that store visual design attributes.<br/> _— Salesforce UX, Lightning Design System_**

## Usage

<br />

### Style Props

Any `token` can be used with its associated `scale` by using a style prop. Style props are component props which names correlate to CSS properties.
Some style props can be used with shorthand syntax and all props follow longhand syntax.

```jsx
// style prop usage
<Box p={5} m={2} backgroundColor="color-background"></Box>
```

### Getters

If you want to use a `token` outside of style props, you can use the `getToken` style function or `useToken` hook.
The [getToken](#gettoken) style function was built so tokens can be grabbed from components without requiring the use of style props.

```jsx
import styled from '@real-system/styling';
import { getToken, useToken } from '@real-system/theme';
import { Box } from '@real-system/box';

const StyledBox = styled(Box)`
  margin: ${getToken(2, 'space')};
  background-color: ${getToken('color-background')};
`;

// or
const MyComponent = () => {
  const bg = useToken('color-background');
  const space = useToken(5, 'space');
  const styles = {
    backgroundColor: bg,
    margin: space,
  };
  return <div styles={styles}>...</div>;
};
```

#### `getToken` signature

`getToken` uses lodash's `get` and accepts type-checked `token`, `scale` and `fallback` arguments.

```typescript
type ReturnValue = string | number | undefined;
/**
 * A styleFn to get theme tokens
 */
function getToken(
  token: ThemeTokens,
  scale: ThemeScales = 'colors',
  fallback: ReturnValue = undefined
): ReturnValue;
```

#### `useToken` signature

`useToken` uses `getToken` and `useTheme` under the hood and accepts type-checked `token`, `scale` and `fallback` arguments.

```typescript
type ReturnValue = string | number | undefined;
/**
 * A styleFn to get theme tokens
 */
function useToken(
  token: ThemeTokens,
  scale: ThemeScales = 'colors',
  fallback: ReturnValue = undefined
): ReturnValue;
```
