## Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom 
# yarn
$ yarn add react react-dom 

# install react-aria

# npm
$ npm install @real-system/a11y-library
# yarn
$ yarn add @real-system/a11y-library
```
