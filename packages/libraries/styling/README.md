## Usage

### Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom
# yarn
$ yarn add react react-dom

# install styling

# npm
$ npm install @real-system/styling
# yarn
$ yarn add @real-system/styling
```

### Code Example

```jsx
import styled from '@real-system/styling`;

const Flex = styled.div`
  display: flex;
`;

const MyComponent = () => {
  return (
    <Flex>
      ...
    </Flex>
  );
};

```
