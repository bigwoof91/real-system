# `@realsystem/styling`

## Usage

### Installation

```bash
# install peer dependencies

# npm
$ npm install react react-dom
# yarn
$ yarn add react react-dom

# install styling

# npm
$ npm install @realsystem/styling
# yarn
$ yarn add @realsystem/styling
```

### Code Example

```jsx
import styled from '@realsystem/styling`;

const Flex = styled.div`
  display: flex;
`;

const MyComponent = () => {
  return (
    <Flex>
      ...
    </Flex>
  );
};

```
